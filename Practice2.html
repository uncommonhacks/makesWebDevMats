Three ways to call JS

  • script tag in the head
  • script tag in the body
  • external .js files

JavaScript Operators

  • + : Add
    Example: x + y =

  • - : Subtract
    Example: x - y =

  • * : Multiply
    Example: x * y =

  • / : Divide
    Example: x / y =

  • % : Modulus (finds the remainder from division)
    Example: x % y =

  • ++ : Incrementation
    Example: x++, ++x, y++, ++y =

  • -- : Decrementation
    Example: x--, --x, y--, --y =

  • You can also put an "=" after any operator to make it an ASSIGNMENT operator!

JavaScript Numbers

Unlike with other languages, JS only has ONE type of Number: 64 bit Floating points! Technically you can also have integer numbers (so no decimals) but they still "count" as being 64-bit floats!

  • 0 - 51: Number
  • 52 - 62: Exponent
  • 63 : Sign

On a side but still related note, JS supports built in math properties!

JavaScript Booleans

As with other languages, JS booleans have two possible values - true or false!!

    Conditional Operators - return whether the operation is true or false
  • == : equal to
  • === : equal to and equal value type
  • != : Not equal
  • !== : NOT equal to nor equal value type
  • > : greater than
  • < : less than
  • >= : greater than or equal
  • <= : greater than or equal
    Logical Operators - combine conditional operators!
  • && : and
  • || : or
  • ! : not
